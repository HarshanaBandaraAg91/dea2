# 🏨 Hotel Management System - Spring Boot Microservices + React Web App

A complete **Hotel Management System** built using:

✅ **Spring Boot Microservices Architecture**  
✅ **React.js Web Frontend Dashboard**  

This project is based on the final ER Diagram and supports hotel operations such as:

- Room Booking & Reservations  
- Guest Management & Loyalty Points  
- Billing & Payments  
- Restaurant & Room Service Orders  
- Spa & Pool Packages  
- Event Hall Booking & Equipment Allocation  
- Inventory & Supplier Stock Management  
- Housekeeping & Cleaning  
- Employee Shift Assignments  
- User Authentication with JWT + Roles & Permissions
